如何用 Markdown 写公众号文章

在 Markdown 介绍文章 使用 Markdown 进行高效写作 中，我们得以见识 Markdown 用于排版的便捷性。假如我想用它来写微信公众号，该怎么操作呢？

我们都知道，微信公众号用的是富文本编辑器，是无法直接解析 Markdown 语法的。但我们可以自己先将 Markdown 解析为富文本，再粘贴进公众号文章的编辑界面。

MD2WeChat#

MD2WeChat 是我根据开源项目 lyricat/wechat-format 定制的一个网页工具，可以很方便地将 Markdown 语法解析为富文本：

将 Markdown 语法的文章粘贴到左栏，并在右栏点击一键复制，粘贴进公众号的编辑器即可。

注：因公众号文章中不允许有外部链接，所以外链会被自动加脚注，并在底部附上。
